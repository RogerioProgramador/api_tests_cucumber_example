# TESTE API Cucumber

Projeto apresentado e feito durante as aulas do QA Academy do Robson Agapito.


## PRÉ-REQUISITOS

Requisitos de software e hardware necessários para executar este projeto de automação

*   Java 1.8 SDK
*   Maven 3.5.*
*   Intellij IDE
*   Plugins do Intellij
    * Cumcuber for java + Gherkin
    * Lombok
    * Ideolog
